# BuzzPit 
## e-commerce for sport products
### Used Technologies 
This is a project for university, it consist in an e-commerce build with the MERN Stack approach:
-  MongoDB 🍃
-  Express.js 📶
-  React.js ⚛️
-  Node.js 🔗

### What a user cand do 
The customer can buy sport products realted to three categories:
-  running 🏃
-  hiking ⛰️
-  swimming 🏊
